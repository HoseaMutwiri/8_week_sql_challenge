# 📊 8-Week SQL Challenge Solutions

Welcome to my repository dedicated to the **[8-Week SQL Challenge](https://8weeksqlchallenge.com/)** created by Danny Ma. This repository serves as a portfolio of my advanced SQL capabilities, data manipulation techniques, and structural database problem-solving skills.

---

## 🛠️ Tech Stack & Skills Demonstrated
* **Database Engine:** PostgreSQL
* **Database Design:** Entity-Relationship (ER) Modeling.
* **Python3:** pandas,SQLAIchemy & psycopg 
* **SQL Complexities:** Common Table Expressions (CTEs), Window Functions (`RANK`, `DENSE_RANK`, `LEAD/LAG`), Complex Table Joins, Subqueries, Aggregate Performance, Data Cleaning & Type Casting.
